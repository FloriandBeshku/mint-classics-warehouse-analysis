# Mint Classics Co. — Inventory Optimization & Supply Chain Consolidation Study

## 📌 Project Overview
This business case study focuses on data-driven supply chain optimization, inventory management, and relational database analytics. Acting as a Data Analyst for Mint Classics, the primary objective of this project is to evaluate an enterprise warehouse network holding over 550,000 physical product items split across distinct global fulfillment locations. 

Through structural exploratory data analysis (EDA) and quantitative stress-testing via SQL, this study investigates whether the company can strategically close an underutilized, inefficient storage node to cut corporate operational overhead while keeping its signature customer delivery speeds 100% uncompromised.

## 🛠️ Tech Stack & Local Environment
* **RDBMS Engine:** MySQL Server 8.0
* **Database Management Client:** MySQL Workbench
* **Analytical Core Language:** Pure Relational SQL (Aggregations, Correlated Subqueries, Left Joins)
* **Portfolio Assets:** Fully assembled, paragraph-mode executive document (.pdf)
